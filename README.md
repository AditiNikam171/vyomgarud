### Design Philosophy

The design follows a **military-industrial aesthetic** with modern web design principles:

1. **Dark & Professional** - Charcoal backgrounds (#1a1a1a, #0f0f0f) convey sophistication and technical prowess
2. **High Contrast** - White text on dark backgrounds ensures readability
3. **Orange Accents** - Strategic use of #ff7b00 for CTAs and highlights creates visual hierarchy
4. **Minimalist Layout** - Clean, spacious design lets content breathe
5. **Subtle Motion** - Animations are purposeful, not distracting

#### Color Palette

Primary Background: #1a1a1a (Charcoal)
Secondary Background: #0f0f0f (Darker Charcoal)
Primary Text: #ffffff (White)
Secondary Text: #d1d5db (Light Gray)
Tertiary Text: #9ca3af (Medium Gray)
Brand Color: #ff7b00 (Vibrant Orange)
Accent Hover: #e66d00 (Darker Orange)

#### Layout Strategy
- **Container Max-Width:** 1280px (optimal reading width)
- **Responsive Breakpoints:**
  - Mobile: < 480px
  - Tablet: 481px - 768px
  - Desktop: 769px - 1024px
  - Large Desktop: > 1024px
