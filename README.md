A modern, responsive landing page for VyomGarud, a military-grade UAV/drone systems company. Built using pure HTML, CSS, and JavaScript, featuring a dark, professional design inspired by aerospace technology.

![VyomGarud Landing Page](https://images.unsplash.com/photo-1473968512647-3e447244af8f?w=1200&q=80)


Features-
Smooth scroll navigation
Scroll-triggered animations (Intersection Observer API)
Parallax hero section
Hover glow effects
Functional contact form with validation
Mobile-first responsive design

Setup Instructions-
1.Clone Repository
  git clone <your-repo-url>
  cd vyomgarud

2.Open Locally-
Simply open index.html
